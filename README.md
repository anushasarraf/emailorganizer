# Job Email Organizer

A dashboard to organize all the job application related emails you get into different categories: 
1) applied
2) rejected
3) next steps
4) offer

## Current progress:
Successful implementation of spam detector

## Motivation:
I am tired of applying to jobs and keeping track of them.
